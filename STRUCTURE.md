### Структура файлов проекта

```
/flask_rl-main
│── app.py                 # Основной файл Flask-приложения
│── config.py              # Файл конфигурации (если требуется)
│── models.py              # Модели базы данных (если используются)
│── requirements.txt       # Список зависимостей Python
│── heroes.json            # Данные о героях Dota 2 (популярность, win rate)
│
├── /static                # Статические файлы (CSS, изображения)
│   ├── styles.css         # Стили для сайта
│
├── /templates             # HTML-шаблоны
│   ├── index.html         # Главная страница с рейтингами героев
│
└── /venv                  # Виртуальное окружение (создаётся при установке)
```

Эта структура поможет понять, где что находится в проекте.
